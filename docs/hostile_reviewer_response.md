# Hostile Reviewer Response

The strongest criticism is correct: an embodied audit only works when its physical probe actually disrupts the shortcut channel. V2 hardening shows false certification returns under weak probe coupling.

The revised paper therefore makes a narrower claim. It argues for a reporting contract: identify suspected shortcut channels, physically perturb them, verify probe coverage, and report false certification and false rejection. The current evidence is suitable for a workshop diagnostic note, not a universal benchmark or safety guarantee.
