# Deception-Resistant Embodied Evaluation for Robots

Submission-hardening version: v2.

Decision: workshop-only.

The paper has a useful diagnostic thesis: robot evaluation should report false certification and physically perturb suspected shortcut channels. The v2 hardening pass narrows the claim. When probe-shortcut coupling is full, embodied audit false certification is 0.004. When coupling drops to 0.25, false certification rises to 0.105; with no effective coupling, it rises to 0.238.

This repository should not claim a universal benchmark or deployment guarantee. The supported contribution is a workshop-level diagnostic and reporting contract: identify shortcut hypotheses, perturb them physically, verify probe coverage, and report false certification.
